# Logseq Text Toolkit UI 设计

## 设计概述

基于用户提供的截图，设计一个增强选中文字编辑体验的 Logseq 插件，包括工具栏和弹窗组件。

## 工具栏设计

### 样式参考

![工具栏样式参考](https://i.imgur.com/your-image-link.png)  <!-- 这里应该使用实际的截图链接 -->

### 设计规范

1. **布局**：
   - 水平排列的工具栏，位于选中文本的上方
   - 圆角矩形设计，带有轻微的阴影
   - 工具栏项之间有适当的间距

2. **颜色**：
   - **Light 主题**：
     - 背景：#ffffff
     - 边框：#e0e0e0
     - 文字：#333333
     - 图标：#666666
     - 悬停背景：#f5f5f5
   - **Dark 主题**：
     - 背景：#333333
     - 边框：#555555
     - 文字：#ffffff
     - 图标：#cccccc
     - 悬停背景：#444444

3. **组件**：
   - 工具栏项：包含图标和可选的标签
   - 分组项：支持下拉菜单
   - 分隔线：用于分隔不同功能组

4. **交互**：
   - 悬停效果：轻微的背景色变化
   - 点击效果：短暂的按压动画
   - 分组项：悬停时显示下拉菜单

## 功能设计

### 工具栏功能

1. **文本格式**：
   - 加粗：将选中文字变为 `**文字**`
   - 斜体：将选中文字变为 `*文字*`
   - 删除线：将选中文字变为 `~~文字~~`

2. **高亮**：
   - 黄色高亮：将选中文字变为 `==文字==`
   - 其他颜色高亮：通过下拉菜单选择

3. **链接**：
   - 文件链接：将选中文字变为 `[[文字]]`
   - 网页链接：打开弹窗输入 URL

4. **评论/注解**：
   - 打开弹窗输入评论内容
   - 将评论数据插入到当前页面或 journal

### 弹窗功能

1. **评论弹窗**：
   - 标题："添加评论"
   - 内容：多行文本输入框
   - 操作：取消、确定

2. **链接弹窗**：
   - 标题："添加链接"
   - 内容：单行文本输入框，用于输入 URL
   - 操作：取消、确定

## 响应式设计

- **桌面端**：完整显示所有工具栏项
- **平板端**：可能需要折叠部分工具栏项到下拉菜单
- **移动端**：只显示最常用的工具栏项，其余通过下拉菜单访问

## 无障碍设计

- 支持键盘导航
- 支持屏幕阅读器
- 适当的颜色对比度
- 清晰的视觉反馈

## 实现建议

1. **组件化**：
   - 将工具栏和弹窗拆分为独立的组件
   - 使用 React 组件化开发

2. **样式管理**：
   - 使用 CSS Modules 或 styled-components
   - 支持主题切换

3. **状态管理**：
   - 使用全局状态管理工具
   - 管理工具栏的可见性和位置

4. **性能优化**：
   - 延迟加载非关键组件
   - 优化渲染性能

## 总结

本设计文档提供了 Logseq Text Toolkit 插件的 UI 设计规范，包括工具栏和弹窗的样式、布局和交互。通过遵循这些设计规范，可以创建一个美观、易用的插件界面，增强用户的文字编辑体验。
