# 更新日志 - 2026-04-23

## 功能优化

1. **正则表达式优化**
   - 简化了 `remove-formatting` 功能的正则表达式，提高了处理效率
   - 修复了正则替换时出现 `$7$8` 等未定义捕获组的问题

   - 移除了设置界面中的宽度配置项，简化了设置选项

4. **错误处理优化**
   - 将所有 `console.error` 改为 `console.warn`，减少错误日志的干扰
   - 确保所有错误都能被正确捕获和处理
