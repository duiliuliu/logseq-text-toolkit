# 错题本

## 2026-04-26

### 问题描述 1
在 TextSelectionDemo 组件中，`<ul>` 元素被直接放在了 `<p>` 元素内部，违反了 HTML 规范，导致 React 抛出 validateDOMNesting 警告。

### 问题原因 1
HTML 规范中，`<ul>` 元素不能作为 `<p>` 元素的子元素。当 React 检测到这种无效的 DOM 嵌套时，会抛出警告。

### 解决方案 1
将 `<ul>` 元素从 `<p>` 元素中移出，作为 `<div>` 的直接子元素，确保 DOM 结构符合 HTML 规范。

### 影响范围 1
测试页面的 DOM 结构，可能影响页面渲染和可访问性。

### 问题描述 2
在使用 InlineCommentModal 时，出现 "选中的文字在块内容中未找到" 错误，导致评论功能无法正常工作。

### 问题原因 2
1. `findAndReplaceText` 函数在找不到选中的文本时会抛出错误，导致整个更新过程失败。
2. `getCurrentBlock` 方法可能会错误地获取到评论模态框的内容，而不是实际选中的文本所在的块内容。

### 解决方案 2
1. 修改 `findAndReplaceText` 函数，使其在找不到选中的文本时返回原始内容，而不是抛出错误。
2. 修改 `updateBlockContent` 函数，只有当内容确实发生变化时才更新块。
3. 修改 `getCurrentBlock` 方法，优先从 `main-content-container` 元素中获取内容，避免获取到评论模态框的内容。

### 影响范围 2
行内评论功能，影响用户添加评论的体验。
