
# Logseq Text Toolkit v1.1.2 发布说明

## 🐛 修复
- **修复嵌套格式处理问题**：修复了包含格式标记的混合内容被错误包裹引号的问题
- **优化引号包裹逻辑**：当文本包含 hiccup 格式片段时，不会在整个字符串外层错误地包裹引号

## 📝 详细改进

### 核心修复
在 `src/lib/textReplace/utils.ts` 中进行了两项关键改进：

1. **`wrapWithQuotesIfNeeded` 函数**：
   - 新增检测：如果文本包含 hiccup 格式片段（`[:`），不应该包裹引号
   - 这样可以避免已处理过的嵌套格式被再次错误地包裹

2. **`handleNestedQuotes` 函数**：
   - 新增 `nestedContainsHiccup` 检查
   - 当处理后的文本包含任何 hiccup 片段时，直接返回而不调用引号包裹函数
   - 更智能地判断何时应该应用引号包裹

### 示例场景修复
**修复前**（错误）：
```
输入: 数组越大 → **扫描越久 → STW（暂停时间）越长 → 服务卡顿**。
输出: [:span.blue "数组越大 → [:b \"扫描越久 → STW（暂停时间）越长 → 服务卡顿\"]。"]
```

**修复后**（正确）：
```
输入: 数组越大 → **扫描越久 → STW（暂停时间）越长 → 服务卡顿**。
输出: [:span.blue 数组越大 → [:b "扫描越久 → STW（暂停时间）越长 → 服务卡顿"]。]
```
