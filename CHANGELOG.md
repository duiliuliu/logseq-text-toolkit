# 更新日志

## v1.0.3 (2026-05-07)

### ✨ 新增功能

**任务进度追踪增强**
- 支持嵌套任务统计，可设置统计层级（1-N 层或所有层级）
- 新增"仅统计叶子任务"选项，忽略父任务只统计最末端子任务
- 实时响应设置变化，UI 自动更新

**状态自动识别优化**
- 任务状态大小写不敏感匹配（支持 Done/done/DONE）
- 支持自定义状态颜色配置

### ⚡ 性能优化

**React DOM Root 复用**
- 使用 Map 结构管理多个容器实例
- 避免重复创建 React Root，提升渲染性能
- 减少内存占用，支持多组件同时渲染

**统一状态管理**
- 简化 Settings API 设计
- 自动响应系统主题和语言设置
- 实时监听设置变化

### 🔧 修复问题

- 修复 TaskProgress 百分比显示为 0% 的问题
- 修复导入路径文件扩展名问题
- 修复设置面板更新后展示不刷新的问题

### 📁 文件结构优化

- 将 `src/lib/initializer.ts` 迁移到 `src/initializer.ts`
- 统一初始化流程管理
